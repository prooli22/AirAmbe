

----- Explications du contenu de la Version 1.0 ----
	
	AirAmbe.exe est l'application principale.
	Generateur.exe est l'application qui génère le fichier "scenarios.txt".
	
	Dans le fichier AirAmbe.sql on retrouve un script sql qui crée et insert les données de notre Base de Données.


---- Étape 1 : Mettre en ligne la Base de Donnée ----

	Pour mettre en fonction la Base de Données, vous pouvez exécuter le script sous un serveur : localhost. Le script créer la Base de Données du nom : AirAmbe.
	
	Pour vous y connecter vous pouvez utiliser les informations (Server=localhost;userid=root;password=;database=AirAmbe), celles-ci sont déjà préconfigurée
	dans les fichiers AirAmbe.exe.config et Generateur.exe.config.	

---- Étape 2 : Générer un fichier scenarios.txt ----

	Afin d'utiliser notre application principale, vous devez tout d'abord créer un fichier de scénario à l'aide du Générateur.
	
---- Étape 3 : Utiliser l'application ----

	Une fois le scénario créer vous pouvez vous amuser dans notre simulateur de vols à la fine pointe de la technologie.
